# BMB 美国用户端

## 技术选型
1. React
2. React-router
3. Mobx
4. Typescript
5. antd

## 启动

```shell
npm install / yarn add
```

```shell
npm start / yarn start
```

## 构建 Staging／uat
提交 pr 到 staging ，合并后 CI 自动部署

## 发布
提交 pr 由 Master 分支 到 release 分支，合并后由 CI 调用 shipit 自动部署
